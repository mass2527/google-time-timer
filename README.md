# Google Time Timer

![test](./public/time-timer.svg)

## How to Use

1. **Setting the Timer:**

   - Click and rotate the timer counterclockwise to set the remaining minutes.

2. **Notification Toggle:**

   - Use the notification button to turn on/off notifications for when the timer ends.
